# AWS DVA-C02 Study Platform 🚀

A free, offline-capable study platform for the **AWS Certified Developer – Associate (DVA-C02)** exam.

🌐 **Live site:** [jlindqvisttt.github.io/aws-dva-c02-study](https://jlindqvisttt.github.io/aws-dva-c02-study/)

---

## Features

- 📝 **471+ practice questions** sourced from multiple exam dumps
- 🎯 **Multiple quiz modes** — Quick Quiz (10/20/30 questions), Exam Simulation (65q, 130min timer), Topic Quiz, Weak Areas
- 📚 **Study Mode** — AWS service reference cards with key facts for each topic
- 🃏 **Flashcard Mode** — flip-card study for active recall
- ℹ️ **Info cards per question** — quick explanation of the concept being tested
- 📊 **Progress tracking** — mastery tracking stored locally in your browser
- 🌙 **Dark mode** — AWS-inspired design (#232F3E + #FF9900)
- 📱 **Mobile friendly** — works on phone and tablet
- ✈️ **Fully offline** — no external dependencies, no CDN
